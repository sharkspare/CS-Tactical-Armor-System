# Docs Index

这个目录放的是和项目结构、硬件资源、系统图示相关的补充文档。

## 文件说明

- [hardware-resources.md](hardware-resources.md)
  详细的硬件引脚占用、模块接线与使用说明。

- [hardware-mermaid.md](hardware-mermaid.md)
  `Mermaid` 版硬件架构图、思维导图和总线规划图。

- [software-mermaid.md](software-mermaid.md)
  `Mermaid` 版软件分层图、主循环职责图和模块关系图。

- [system-mermaid.md](system-mermaid.md)
  `Mermaid` 版系统总览、数据流、受击事件链路和 GPS/动作识别链路图。

## 阅读建议

如果你先想看接线，先读 `hardware-resources.md`。  
如果你先想看系统全貌，先读 `system-mermaid.md`。  
如果你先想继续开发代码，先读 `software-mermaid.md`。  
如果你要做展示或答辩，这几个 `Mermaid` 文件都很适合直接拿去整理成图。
